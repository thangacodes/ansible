Membership and Test operators of Ansible Playbooks:

  * in
  * not in

Test Operators:

* Tests for variables (is denfined and is undefined)
* Tests for strings (strx is lower, strx is upper, strx is string)
* Tests for numbers (numx is divisibleby 5, is odd, is even, is number)
* Tests for paths (pth1 is directory, pth2 is file, pth3 is link, pth1 is exists, pth2 is same_file(pth4))
